# Delventto

Landing page for the Apartamento Delventto vacation rental in Santa Marta, Colombia.

## Technologies

- React 19
- Create React App
- react-icons
- gh-pages

## Build locally

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Deploy to GitHub Pages

```bash
npm run deploy
```

This runs the production build and pushes the `build/` folder to the `gh-pages` branch. The site is available at:

`https://vanessar810.github.io/delventto/`
